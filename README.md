gh-pages for scifabric.com

How to preview locally:

* Install latest ruby, e.g. with rvm
* Install bundler
* bundle install

```
gem install bundler
bundle install
```

Preview page:

```
jekyll serve
```

---

In case of problems:
Clear all your ruby gems and start from beginning

```
gem uninstall -aIx
```

And that's all.
