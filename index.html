What we do

We do projects!

From science, to the humanities, journalism, medicine and beyond, we host projects that harness the power of the crowd to achieve a positive social impact and make the world a better place.

We develop PyBossa – an open source framework that may be used to build and host tailor-made crowdsourced projects.

visit pybossa

We host Crowdcrafting – an easy-to-use platform where ANYONE can create a research project (no code required!).

visit crowdcrafting
