What we do

We do projects!

Science. Humanities. Journalism. Medicine. Social. Projects that aim to have a social impact. Projects that want to make the world a better place. And, above all, projects that harness the power of the crowd.

We develop PyBossa, a unique open source framework that can be used to build and host tailor-made crowdsourced research projects.

visit pybossa.com

We host Crowdcrafting, an easy-to-use platform where ANYONE can create crowdsourced research projects.

visit crowdcrafting.org
